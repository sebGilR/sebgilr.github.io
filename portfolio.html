Collider Game

Live Demo Source Code

Collider Game

This is a 2D endless running game created using ES6 and the game engine Phaser 3. The objective of the game is to collide against other ships to accumulate points.

The difficulty is increased according to the level of the player. When the game is over the score is saved and a list of top-scores is displayed.
